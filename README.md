# GenExpr for VSCode

This is a VS Code extension based on https://github.com/larme/GenExprForVSCode, which was converted from https://github.com/emmanueljourdan/GenExprForSublime. As of June 2026, all 243 Gen operators have hover and completion support based on documentation from Cycling 74’s [website](https://docs.cycling74.com/userguide/gen/_gen_overview/).

To use this extension, move this folder to the `<user home>/.vscode/extensions` folder and restart VS Code.
